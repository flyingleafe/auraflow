# Example data

- `speech.wav` — 1.3 s clean speech utterance (44.1 kHz, mono, int16), extracted
  from the **DREGON** dataset's clean source recordings
  (`DREGON_clean_recordings_speech/45_0_1.2__1.wav`, loudest 4 s window,
  downmixed). DREGON: "DREGON: Drone Ego-Noise and Localization" dataset,
  released for research use — see https://dregon.inria.fr. Used here solely as
  a small real-speech driver signal for `scripts/speaker_demo.py --wav`.
